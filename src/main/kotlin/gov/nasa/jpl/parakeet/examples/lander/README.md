# Parakeet Lander

Replicates [Aerie Lander](https://github.com/NASA-AMMOS/aerie-lander) in Parakeet more or less verbatim.

This isn't meant to be the best way to write that model in Parakeet, only to be a highly faithful translation.

This can serve as a performance and correctness test of Parakeet against Aerie.
